# Sprint 1 Meta-Freigabe — Startseite

## Ziel

Meta Title und Meta Description der Startseite passend zur neuen Positionierung setzen.

## Vorschlag Meta Title

PV-Blechziegel aus Aluminium für Photovoltaik-Montage

## Vorschlag Meta Description

PV-Blechziegel aus Aluminium für die sichere Montage von Photovoltaikanlagen auf Ziegeldächern. Ersatzdachziegel für Dachhaken, passend für viele Profile.

## Strategische Hinweise

- Keine aktive Bewerbung von Solarziegeln oder Solardachziegeln.
- Keine negative Abgrenzung in der Meta Description.
- Positive Positionierung auf PV-Blechziegel / Aluminium-Ersatzdachziegel / Photovoltaik-Montage.
- Meta-Daten sind separat freizugeben.
- Noch keine Shopify Admin API Writes in diesem Schritt.

## Umsetzung nach Freigabe

Option A:
Manuell im Shopify Admin unter Online Store / Preferences setzen.

Option B:
Separater kontrollierter Admin-API-Schritt, nur nach ausdrücklicher Freigabe.
