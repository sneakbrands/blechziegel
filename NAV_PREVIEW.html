UX-Prinzipien Mobile

Flache Hierarchie (1 Tiefe max), Dachtypen direkt angeklappt sichtbar, CTA unten am Drawer-Ende — genau dort wo der Daumen ruht.

  • Klickpfad: Burger → Dachtyp → Produkt = 3 Taps (statt heute 4–5)
  • CTA "Jetzt Blechziegel finden" ist persistent, führt auf Bestseller-Kollektion
  • Keine verschachtelten Accordions → schneller erfassbar
  • Announcement-Bar bleibt sichtbar über dem Logo (Trust)
  • Suche und Cart rechts im Header für Daumen-Reichweite
Umsetzung · Checklist

So wird das Live 4 Admin-Tasks · 0 Code-Eingriffe

① Menü-Umbau

Shopify Admin → Online Store → Navigation → main-menu. Struktur aus dem Vorschau-Header exakt nachbauen. Submenu "Dachtypen" mit 5 Collections.

② Menu-Style prüfen

In header-group.json ist menu_style: "featured_products" aktiv. Für reines Dropdown ohne Bilder: auf "default" umstellen via Theme-Editor.

③ "Für Dachdecker"-Page

Shopify Admin → Pages → Neue Page "Für Dachdecker" (/pages/gewerbe). Mit B2B-Content befüllen (Mengenpreise, Schnellbestellung).

④ Header-CTA

Als letzter Menüeintrag "Jetzt kaufen" anlegen → /collections/frankfurter-pfanne. Falls visuelle Hervorhebung gewünscht: kleines CSS-Snippet, ich baue es bei Bedarf.
